# 자동차 총비용 비교 프로젝트

## 1. 프로젝트 개요

이 프로젝트는 자동차를 구매하거나 이용한 뒤, 몇 년이 지난 시점에서 어떤 선택이 더 저렴한지 비교하기 위한 프로젝트입니다.

자동차 구매 또는 이용 후 일정 기간 동안 발생하는 구매 비용, 유류비, 보험료, 소모품 비용 등을 종합적으로 계산하여 나에게 가장 저렴하고 적합한 자동차 구매 방법을 찾는 것을 목표로 합니다.

## 2. 프로젝트 주제

자동차 관련 총비용 비교

## 3. 비교 대상

프로젝트에서 비교하는 비용 항목은 다음과 같습니다.

1. 자동차 구매 가격
2. 자동차 유류비
3. 자동차 보험료
4. 일정 기간 동안 발생하는 소모품 비용

## 4. 조사 범위

### 4.1 자동차 회사

- 현대자동차
- 기아자동차

### 4.2 자동차 카테고리

- 승용차

### 4.3 분류

- 신차
- 중고차
- 리스
- 렌트

## 5. 주요 기능

1. 차량 정보 조회
2. 신차/중고차/리스/렌트 구매 비용 비교
3. 예상 유류비 계산
4. 예상 보험료 계산
5. 소모품 비용 계산
6. 일정 기간 이후 총비용 비교

## 6. 분석 기준

자동차를 일정 기간 동안 이용한다고 가정하고, 다음 항목을 합산하여 총비용을 계산합니다.

```text
총비용 = 차량 가격 + 유류비 + 보험료 + 소모품 비용
```

## 7. 기대 효과

사용자는 자동차 구매 또는 이용 후 몇 년이 지난 시점에서 발생하는 전체 비용을 비교할 수 있습니다.

단순히 차량 가격만 비교하는 것이 아니라, 일정 기간 동안의 유지비까지 포함하여 어떤 자동차 이용 방식이 더 저렴한지 판단할 수 있습니다.
