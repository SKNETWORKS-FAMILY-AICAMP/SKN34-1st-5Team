# XML CSV 변환

## 목표

`XML` API 또는 XML 파일을 읽어서 분석하기 쉬운 `CSV` 파일로 변환한다.

공공데이터 API는 XML 형태로 응답하는 경우가 많다. XML은 태그로 데이터를 감싸는 구조라서, 반복되는 기준 태그를 찾고 그 안에 있는 하위 태그 값을 꺼내면 표 형태로 만들 수 있다.

XML CSV 변환은 보통 아래 흐름으로 진행한다.

```text
XML API 주소 또는 XML 파일 준비
→ requests로 XML 문서 가져오기
→ BeautifulSoup xml 파서로 파싱하기
→ 반복 기준 태그 찾기
→ 하위 태그 값을 한 줄씩 추출하기
→ pandas DataFrame으로 만들기
→ CSV 파일로 저장하기
```

## 최종 산출물

- XML API URL 또는 XML 파일
- 반복 기준 태그
- 추출할 태그 목록
- 전처리 완료된 `CSV` 파일
- 데이터 출처와 수집 날짜 기록

## 작업 순서

1. XML URL 또는 XML 파일 경로를 준비한다.
2. XML 구조를 열어보고 반복 기준 태그를 확인한다.
3. 추출할 하위 태그 목록을 정한다.
4. `requests`로 XML 데이터를 가져온다.
5. `BeautifulSoup(response.text, "xml")`로 XML을 파싱한다.
6. `find_all()`로 반복 기준 태그를 전부 찾는다.
7. `find_text()`로 하위 태그 값을 안전하게 추출한다.
8. 추출 결과를 `pandas DataFrame`으로 만든다.
9. 결측치, 중복값, 컬럼명을 정리한다.
10. 최종 결과를 `CSV`로 저장한다.

## 1. XML이란?

XML은 데이터를 태그로 감싸서 표현하는 형식이다.

예시 XML:

```xml
<items>
  <item>
    <name>서울</name>
    <value>10000</value>
    <year>2024</year>
  </item>
  <item>
    <name>부산</name>
    <value>5000</value>
    <year>2024</year>
  </item>
</items>
```

이 XML에는 `item` 태그가 반복되고 있다.

첫 번째 `item` 데이터:

```text
name: 서울
value: 10000
year: 2024
```

두 번째 `item` 데이터:

```text
name: 부산
value: 5000
year: 2024
```

CSV로 바꾸면 아래처럼 된다.

```csv
name,value,year
서울,10000,2024
부산,5000,2024
```

### 한 줄씩 설명

`"xml"` 파서를 사용하면 XML 태그 구조에 맞게 파싱할 수 있다.

## 4. 반복 기준 태그 찾기

XML을 CSV로 바꿀 때 가장 중요한 것은 반복되는 기준 태그를 찾는 것이다.

예시 XML:

```xml
<item>
  <name>서울</name>
  <value>10000</value>
</item>
<item>
  <name>부산</name>
  <value>5000</value>
</item>
```

여기서는 `item`이 한 행을 의미한다.

```python
items = soup.find_all("item")
```

결과 예시:

```python
[
    <item><name>서울</name><value>10000</value></item>,
    <item><name>부산</name><value>5000</value></item>
]
```

`items`에는 `item` 태그들이 리스트처럼 들어간다.

## 5. 하위 태그 값 추출하기

반복 기준 태그 안에서 필요한 값을 꺼낸다.

```python
for item in items:
    name = item.find_text("name")
    value = item.find_text("value")
```

### 한 줄씩 설명

```python
for item in items:
```

`items`에 들어있는 `item` 태그를 하나씩 꺼낸다.

첫 번째 반복의 `item`:

```xml
<item>
  <name>서울</name>
  <value>10000</value>
</item>
```

```python
name = item.find_text("name")
```

현재 `item` 안에서 `name` 태그의 텍스트를 꺼낸다.

결과 예시:

```text
서울
```

```python
value = item.find_text("value")
```

현재 `item` 안에서 `value` 태그의 텍스트를 꺼낸다.

결과 예시:

```text
10000
```

`find_text()`를 사용하면 태그가 없을 때도 비교적 안전하게 처리할 수 있다.

## 6. rows 리스트 만들기

CSV로 저장하려면 행 단위 딕셔너리 리스트를 만드는 것이 좋다.

rows = []

이 구조는 바로 `pandas DataFrame`으로 변환하기 쉽다.

## 8. XML 파일을 직접 읽는 경우

API가 아니라 로컬 XML 파일을 읽는 경우도 있다.

```python
from bs4 import BeautifulSoup

file_path = "data/raw/source.xml"

with open(file_path, "r", encoding="utf-8") as file:
    xml_text = file.read()

soup = BeautifulSoup(xml_text, "xml")
```

`file_path`에는 로컬 XML 파일 경로가 들어간다.

`file.read()`는 파일 내용을 문자열로 읽어온다.

## 9. 최종 예시 코드

```python
import requests
from bs4 import BeautifulSoup
import pandas as pd

url = "XML API URL"

response = requests.get(url)
response.raise_for_status()

xml_text = response.text
soup = BeautifulSoup(xml_text, "xml")

items = soup.find_all("item")
print("찾은 item 개수:", len(items))

rows = []

for item in items:
    rows.append({
        "name": item.find_text("name"),
        "value": item.find_text("value"),
        "year": item.find_text("year"),
    })

df = pd.DataFrame(rows)

df.columns = df.columns.astype(str).str.strip()
df = df.dropna(how="all")
df = df.drop_duplicates()

print(df.head())
print(df.info())
print(df.isnull().sum())
print(df.duplicated().sum())

df.to_csv("data/processed/xml_data.csv", index=False, encoding="utf-8-sig")
```

## 11. 자주 나는 오류

### XML 태그를 찾지 못하는 경우

```python
items = soup.find_all("item")
print(len(items))
```

결과가 `0`이면 반복 기준 태그 이름이 다를 수 있다.

해결:

- XML 원문을 출력해서 실제 태그명을 확인한다.
- `item`, `row`, `data`, `record` 등 반복되는 태그명을 찾는다.

### XML 파서 오류

`BeautifulSoup(xml_text, "xml")`에서 문제가 생기면 `lxml` 설치가 필요할 수 있다.

```bash
pip install lxml
```

### API 인증 오류

공공데이터 API는 인증키가 필요할 수 있다.

확인할 것:

- API 키가 URL에 들어갔는지
- API 키가 인코딩되어야 하는지
- 요청 제한 횟수를 초과하지 않았는지
- 응답 메시지에 에러 코드가 있는지

## 12. 기본 코드 템플릿

실제 프로젝트에서는 아래 코드에서 `url`, `item`, 추출할 태그명만 바꿔서 사용하면 된다.

```python
import requests
from bs4 import BeautifulSoup
import pandas as pd

url = "XML API URL"
response = requests.get(url)
response.raise_for_status()

soup = BeautifulSoup(response.text, "xml")

rows = []
for item in soup.find_all("item"):
    rows.append({
        "name": item.find_text("name"),
        "value": item.find_text("value"),
    })

df = pd.DataFrame(rows)
df.columns = df.columns.astype(str).str.strip()
df = df.dropna(how="all")
df = df.drop_duplicates()

df.to_csv("data/processed/xml_data.csv", index=False, encoding="utf-8-sig")
```

## 초보자가 꼭 기억할 것

- XML은 태그로 데이터를 감싸는 구조다.
- 반복되는 태그 하나가 CSV의 한 행이 되는 경우가 많다.
- `BeautifulSoup(xml_text, "xml")`로 XML을 파싱한다.
- `find_all("item")`은 모든 `item` 태그를 찾는다.
- `find_text("name")`은 현재 태그 안의 `name` 값을 꺼낸다.
- 추출한 값은 딕셔너리 리스트로 만든 뒤 `DataFrame`으로 변환한다.
- 최종적으로 `to_csv()`로 저장한다.

## 완료 기준

- 반복 기준 태그에서 데이터가 정상 추출된다.
- 태그가 없는 데이터가 있어도 코드가 중단되지 않는다.
- `data/processed/xml_data.csv` 파일이 생성된다.
- 결측치와 중복값을 확인했다.
- 수집 대상 URL 또는 파일 경로와 수집 날짜가 기록되어 있다.
