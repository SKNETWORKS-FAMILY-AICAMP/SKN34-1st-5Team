# HTML table CSV 변환

## 목표

웹페이지 안에 있는 `HTML table` 데이터를 읽어서 `CSV` 파일로 저장한다.

`HTML table`은 웹페이지에서 표 형태로 보이는 데이터다. 예를 들어 지역별 전기차 등록 대수, 연도별 통계, 충전소 현황처럼 행과 열로 정리된 데이터가 웹페이지에 표로 올라와 있다면 `pandas.read_html()`로 쉽게 가져올 수 있다.

HTML table 변환은 보통 아래 흐름으로 진행한다.

```text
표가 있는 웹페이지 주소 입력
→ pandas.read_html()로 페이지 안의 모든 표 가져오기
→ 표 개수와 각 표의 크기 확인하기
→ 사용할 표 선택하기
→ DataFrame 전처리하기
→ CSV 파일로 저장하기
```

## 최종 산출물

- 표가 있는 웹페이지 URL
- 사용할 표 번호 또는 표 설명
- 전처리 완료된 `CSV` 파일
- 데이터 출처와 수집 날짜 기록

## 작업 순서

1. 표가 있는 웹페이지 URL을 정한다.
2. 브라우저에서 해당 페이지에 표가 실제로 있는지 확인한다.
3. `pandas.read_html()`로 페이지 안의 모든 표를 가져온다.
4. 각 표의 번호, 행 개수, 열 개수를 확인한다.
5. 미리보기를 통해 사용할 표를 선택한다.
6. 선택한 표를 `DataFrame`으로 저장한다.
7. 컬럼명, 빈 행, 중복값, 결측치를 정리한다.
8. 최종 결과를 `CSV`로 저장한다.

## 1. HTML table이란?

HTML table은 HTML에서 `<table>`, `<tr>`, `<th>`, `<td>` 태그로 만들어진 표다.

예시 HTML:

```html
<table>
  <tr>
    <th>지역</th>
    <th>전기차 등록대수</th>
  </tr>
  <tr>
    <td>서울</td>
    <td>10000</td>
  </tr>
  <tr>
    <td>부산</td>
    <td>5000</td>
  </tr>
</table>
```

이 표에는 아래 데이터가 들어있다.

```text
컬럼: 지역, 전기차 등록대수
1행: 서울, 10000
2행: 부산, 5000
```

웹페이지에 이런 표가 있으면 `pandas.read_html()`이 자동으로 표를 찾아서 `DataFrame`으로 바꿔준다.

## 2. pandas.read_html로 표 가져오기

`pandas.read_html()`은 웹페이지 안에 있는 `<table>` 태그를 전부 찾아서 리스트로 반환한다.

### 기본 코드

```python
import pandas as pd

url = "표가 있는 URL"
tables = pd.read_html(url)
```

### 한 줄씩 설명

```python
import pandas as pd
```

`pandas` 라이브러리를 가져온다. `pandas`는 표 형태 데이터를 다룰 때 가장 많이 사용한다.

```python
url = "표가 있는 URL"
```

표가 들어있는 웹페이지 주소를 `url` 변수에 저장한다.

예시:

```text
https://example.com/statistics
```

```python
tables = pd.read_html(url)
```

웹페이지 안에 있는 모든 HTML table을 읽어온다.

`tables`에는 표가 여러 개 들어갈 수 있다. 즉, `tables`는 `DataFrame` 하나가 아니라 `DataFrame` 여러 개가 들어있는 리스트다.

결과 예시:

```python
[
    DataFrame_0,
    DataFrame_1,
    DataFrame_2,
]
```

## 3. 표 개수 확인하기

웹페이지 안에는 표가 하나만 있을 수도 있고 여러 개 있을 수도 있다.

```python
print(len(tables))
```

결과 예시:

```text
3
```

이 결과는 해당 페이지에서 표를 3개 찾았다는 뜻이다.

## 4. 각 표의 크기 확인하기

어떤 표를 사용할지 고르기 위해 각 표의 행과 열 개수를 확인한다.

```python
for index, table in enumerate(tables):
    print(index, table.shape)
```

### 한 줄씩 설명

```python
for index, table in enumerate(tables):
```

`tables` 안에 있는 표를 하나씩 꺼낸다.

`index`에는 표 번호가 들어간다.

`table`에는 실제 `DataFrame`이 들어간다.

```python
print(index, table.shape)
```

표 번호와 표 크기를 출력한다.

`table.shape`는 `(행 개수, 열 개수)`를 의미한다.

결과 예시:

```text
0 (5, 3)
1 (17, 4)
2 (3, 2)
```

해석:

```text
0번 표: 5행 3열
1번 표: 17행 4열
2번 표: 3행 2열
```

만약 우리가 찾는 데이터가 17개 시도별 데이터라면 `1번 표`가 후보일 가능성이 높다.

## 5. 표 미리보기

표 번호만 보고 판단하기 어려우면 `head()`로 앞부분을 확인한다.

```python
print(tables[0].head())
print(tables[1].head())
```

`head()`는 표의 앞 5줄을 보여준다.

결과 예시:

```text
   지역  전기차 등록대수  기준연도
0  서울      10000  2024
1  부산       5000  2024
2  대구       3000  2024
```

이 결과를 보고 원하는 표인지 확인한다.

## 6. 사용할 표 선택하기

원하는 표 번호를 찾았다면 해당 표를 `df` 변수에 저장한다.

```python
df = tables[1]
```

이 코드는 `tables` 리스트의 1번 표를 선택한다는 뜻이다.

주의할 점은 파이썬 번호는 0부터 시작한다.

```text
tables[0] → 첫 번째 표
tables[1] → 두 번째 표
tables[2] → 세 번째 표
```

## 7. DataFrame 전처리하기

표를 가져온 뒤 바로 저장하지 말고 기본적인 정리를 한다.

```python
df.columns = df.columns.astype(str).str.strip()
```

컬럼명을 문자열로 바꾸고 앞뒤 공백을 제거한다.

예시:

```text
" 지역 " → "지역"
```

```python
df = df.dropna(how="all")
```

모든 값이 비어 있는 행을 제거한다.

```python
df = df.drop_duplicates()
```

완전히 같은 중복 행을 제거한다.

```python
print(df.info())
```

컬럼명, 데이터 개수, 데이터 타입을 확인한다.

```python
print(df.isnull().sum())
```

컬럼별 결측치 개수를 확인한다.

## 8. 최종 예시 코드

```python
import pandas as pd

url = "표가 있는 URL"

tables = pd.read_html(url)

print("찾은 표 개수:", len(tables))

for index, table in enumerate(tables):
    print(index, table.shape)
    print(table.head())
    print("-" * 50)

selected_table_index = 0

df = tables[selected_table_index]

df.columns = df.columns.astype(str).str.strip()
df = df.dropna(how="all")
df = df.drop_duplicates()

print(df.head())
print(df.info())
print(df.isnull().sum())
print(df.duplicated().sum())

df.to_csv("data/processed/html_table.csv", index=False, encoding="utf-8-sig")
```

## 10. 자주 나는 오류

### No tables found

```text
ValueError: No tables found
```

원인:

- 페이지 안에 실제 `<table>` 태그가 없을 수 있다.
- 화면에는 표처럼 보이지만 `div`로 만든 표일 수 있다.
- 로그인이 필요한 페이지일 수 있다.
- 자바스크립트로 나중에 표가 로딩되는 동적 페이지일 수 있다.

해결:

- 개발자 도구에서 `<table>` 태그가 있는지 확인한다.
- 동적 페이지라면 Selenium 같은 도구가 필요할 수 있다.
- 데이터가 API로 따로 내려오는지 Network 탭에서 확인한다.

### 한글 깨짐

CSV를 엑셀로 열었을 때 한글이 깨지면 저장할 때 `encoding="utf-8-sig"`를 사용한다.

```python
df.to_csv("data/processed/html_table.csv", index=False, encoding="utf-8-sig")
```

## 11. 기본 코드 템플릿

실제 프로젝트에서는 아래 코드에서 `url`과 `selected_table_index`만 바꿔서 사용하면 된다.

```python
import pandas as pd

url = "표가 있는 URL"
tables = pd.read_html(url)

for index, table in enumerate(tables):
    print(index, table.shape)
    print(table.head())

df = tables[0]
df.columns = df.columns.astype(str).str.strip()
df = df.dropna(how="all")
df = df.drop_duplicates()

df.to_csv("data/processed/html_table.csv", index=False, encoding="utf-8-sig")
```

## 초보자가 꼭 기억할 것

- `pandas.read_html(url)`은 웹페이지 안의 HTML table을 전부 가져온다.
- 결과는 `DataFrame` 하나가 아니라 `DataFrame` 리스트다.
- `len(tables)`로 표 개수를 확인한다.
- `table.shape`는 표의 행과 열 개수를 의미한다.
- `tables[0]`은 첫 번째 표를 의미한다.
- `head()`로 표 앞부분을 확인한 뒤 원하는 표를 선택한다.
- 최종적으로 `DataFrame`을 정리한 뒤 `to_csv()`로 저장한다.

## 완료 기준

- 웹페이지의 표가 `DataFrame`으로 정상 변환된다.
- 원하는 표 번호를 확인할 수 있다.
- `data/processed/html_table.csv` 파일이 생성된다.
- 컬럼명, 빈 행, 중복 행, 결측치가 확인되어 있다.
- 수집 대상 URL과 수집 날짜가 기록되어 있다.
