# 환경 세팅
- Docker 설치 권장
- python : 3.12
- mysql : 8.4

## venv 가상환경 세팅

```bash
python -m venv 1st_project
source .venv/bin/activate
pip install -r requirements.txt
```

## Anaconda 가상환경 세팅
```bash
conda create --name 1st_project python=3.12 -y
conda activate 1st_project
pip install -r requirements.txt
```

## Docker MySQL 실행
```bash
docker compose up
```
## 패키지 버전 갱신
```bash
pip freeze > requirements.txt
```
