# 정적페이지 크롤링

## 목표

정적 웹페이지에서 필요한 데이터를 수집한 뒤 `CSV` 파일로 저장한다.

정적 페이지는 브라우저에서 보이는 데이터가 HTML 문서 안에 바로 들어있는 페이지를 말한다. 예를 들어 뉴스 제목, FAQ 질문, 게시글 제목 등이 HTML 코드 안에 이미 들어있다면 정적 페이지로 볼 수 있다.

정적 페이지 크롤링은 보통 아래 흐름으로 진행한다.

```text
웹페이지 주소 입력
→ requests로 HTML 문서 가져오기
→ BeautifulSoup으로 HTML 정리하기
→ find/select로 원하는 데이터만 꺼내기
→ pandas DataFrame으로 만들기
→ CSV 파일로 저장하기
```

## 최종 산출물

- 수집 대상 URL
- 정적 페이지 크롤링 코드
- 전처리 완료된 `CSV` 파일
- 데이터 출처와 수집 날짜 기록

## 작업 순서

1. 수집할 웹페이지 URL을 정한다.
2. 개발자 도구로 필요한 데이터의 HTML 구조를 확인한다.
3. `requests`로 HTML을 가져온다.
4. `BeautifulSoup`으로 HTML을 파싱한다.
5. `find`, `find_all`, `select_one`, `select`로 필요한 태그를 찾는다.
6. `get_text()`로 글자를 꺼내고 `.get("href")`로 링크 같은 속성값을 꺼낸다.
7. 수집 결과를 `pandas DataFrame`으로 만든다.
8. 중복, 결측치, 컬럼명을 정리한다.
9. 최종 결과를 `CSV`로 저장한다.

## 1. requests로 웹페이지 가져오기

`requests`는 파이썬에서 웹페이지에 접속할 때 사용하는 라이브러리다.

설치가 안 되어 있다면 아래 명령어로 설치한다.

```bash
pip install requests
```

### 기본 코드

```python
import requests

url = "https://www.naver.com"
response = requests.get(url)
html_text = response.text
```

### 한 줄씩 설명

```python
import requests
```

`requests` 패키지를 가져온다. 이제 `requests.get()` 같은 기능을 사용할 수 있다.

```python
url = "https://www.naver.com"
```

크롤링할 웹페이지 주소를 `url` 변수에 저장한다.

이때 `url`에 들어있는 데이터는 문자열이다.

```text
https://www.naver.com
```

```python
response = requests.get(url)
```

`url` 주소로 요청을 보낸다. 사람이 브라우저에 주소를 입력해서 접속하는 것처럼, 파이썬이 해당 웹페이지에 접속해서 응답을 받아온다.

`response` 안에는 HTML뿐만 아니라 응답 상태, 최종 URL 등 여러 정보가 함께 들어있다.

예를 들어 아래처럼 확인할 수 있다.

```python
print(response.status_code)
```

결과 예시:

```text
200
```

`200`은 정상 접속을 의미한다.

```python
print(response.url)
```

결과 예시:

```text
https://www.naver.com/
```

최종 접속된 URL을 의미한다.

```python
html_text = response.text
```

응답에서 HTML 문서만 꺼내 `html_text` 변수에 저장한다.

`html_text`에는 아래처럼 웹페이지 전체 HTML 코드가 문자열로 들어간다.

```html
<!doctype html>
<html lang="ko">
<head>
  <title>페이지 제목</title>
</head>
<body>
  <a href="https://example.com">메뉴</a>
</body>
</html>
```

중요한 점은 `html_text`는 아직 태그를 쉽게 찾을 수 있는 상태가 아니라, 그냥 긴 문자열이라는 것이다.

## 2. BeautifulSoup으로 HTML 파싱하기

`BeautifulSoup`은 복잡한 HTML 문자열을 파이썬이 다루기 쉬운 구조로 바꿔주는 라이브러리다.

설치가 안 되어 있다면 아래 명령어로 설치한다.

```bash
pip install beautifulsoup4
```

### 기본 코드

```python
from bs4 import BeautifulSoup

soup = BeautifulSoup(html_text, "html.parser")
```

### 한 줄씩 설명

```python
from bs4 import BeautifulSoup
```

`bs4` 패키지에서 `BeautifulSoup` 기능을 가져온다.

```python
soup = BeautifulSoup(html_text, "html.parser")
```

`html_text`에 들어있는 긴 HTML 문자열을 파싱한다.

파싱 전 `html_text`는 문자열이다.

```text
"<!doctype html><html><head>...</head><body>...</body></html>"
```

파싱 후 `soup`은 태그 단위로 탐색할 수 있는 객체가 된다.

```html
<html>
<head>
<title>페이지 제목</title>
</head>
<body>
<a href="https://example.com">메뉴</a>
</body>
</html>
```

이제 `soup.find()`, `soup.find_all()`, `soup.select()` 같은 기능으로 원하는 태그를 찾을 수 있다.

## 3. find로 태그 하나 찾기

`find()`는 조건에 맞는 태그를 하나만 찾는다.

예시 HTML이 아래와 같다고 가정한다.

```html
<p class="para">코딩유치원</p>
<div id="zara">크롤링 연습</div>
```

이 HTML에는 두 개의 데이터가 있다.

첫 번째 데이터:

```text
태그명: p
class: para
텍스트: 코딩유치원
```

두 번째 데이터:

```text
태그명: div
id: zara
텍스트: 크롤링 연습
```

### 태그명으로 찾기

```python
soup.find("p")
```

결과 예시:

```html
<p class="para">코딩유치원</p>
```

텍스트만 나오는 것이 아니라 태그 전체가 나온다.

### class로 찾기

```python
soup.find(class_="para")
```

결과 예시:

```html
<p class="para">코딩유치원</p>
```

`class`는 파이썬 예약어라서 `class_`처럼 밑줄을 붙여 사용한다.

아래처럼 `attrs`를 사용해도 같은 의미다.

```python
soup.find(attrs={"class": "para"})
```

### id로 찾기

```python
soup.find(id="zara")
```

결과 예시:

```html
<div id="zara">크롤링 연습</div>
```

아래처럼 `attrs`를 사용해도 같은 의미다.

```python
soup.find(attrs={"id": "zara"})
```

### 태그명과 class를 같이 조건으로 사용하기

```python
soup.find("p", class_="para")
```

결과 예시:

```html
<p class="para">코딩유치원</p>
```

조건이 더 구체적이므로 원하는 데이터를 더 정확하게 찾을 수 있다.

## 4. find_all로 태그 여러 개 찾기

`find_all()`은 조건에 맞는 태그를 전부 찾는다.

예시 HTML:

```html
<p class="para">첫 번째 문장</p>
<p class="para">두 번째 문장</p>
<p class="para">세 번째 문장</p>
```

코드:

```python
items = soup.find_all("p")
```

결과 예시:

```python
[
    <p class="para">첫 번째 문장</p>,
    <p class="para">두 번째 문장</p>,
    <p class="para">세 번째 문장</p>
]
```

`items`에는 조건에 맞는 태그들이 리스트처럼 여러 개 들어간다.

각 태그를 하나씩 처리하려면 `for`문을 사용한다.

```python
for item in items:
    print(item.get_text())
```

출력 결과:

```text
첫 번째 문장
두 번째 문장
세 번째 문장
```

## 5. select로 CSS 선택자 사용하기

`select()`는 CSS 선택자를 사용해서 태그를 찾는다.

초보자는 브라우저 개발자 도구에서 CSS 선택자를 복사해서 사용할 수 있어 `select()` 방식이 더 직관적일 수 있다.

### select_one

`select_one()`은 조건에 맞는 태그를 하나만 찾는다.

```python
soup.select_one("a.news_tit")
```

의미:

```text
a 태그 중에서 class가 news_tit인 태그 하나를 찾아줘.
```

예시 HTML:

```html
<a class="news_tit" href="https://news.example.com/1">
  전기차 충전소 확대 발표
</a>
```

결과 예시:

```html
<a class="news_tit" href="https://news.example.com/1">
  전기차 충전소 확대 발표
</a>
```

### select

`select()`는 조건에 맞는 태그를 전부 찾는다.

```python
titles = soup.select("a.news_tit")
```

예시 HTML:

```html
<a class="news_tit">첫 번째 뉴스</a>
<a class="news_tit">두 번째 뉴스</a>
<a class="news_tit">세 번째 뉴스</a>
```

결과 예시:

```python
[
    <a class="news_tit">첫 번째 뉴스</a>,
    <a class="news_tit">두 번째 뉴스</a>,
    <a class="news_tit">세 번째 뉴스</a>
]
```

## 6. get_text로 글자만 가져오기

태그 전체가 아니라 화면에 보이는 글자만 가져오려면 `get_text()`를 사용한다.

```python
for item in titles:
    title = item.get_text()
    print(title)
```

만약 `titles` 안에 아래 데이터가 들어있다면:

```python
titles = [
    <a class="news_tit">첫 번째 뉴스</a>,
    <a class="news_tit">두 번째 뉴스</a>,
    <a class="news_tit">세 번째 뉴스</a>
]
```

첫 번째 반복에서 `item`은 아래 태그다.

```html
<a class="news_tit">첫 번째 뉴스</a>
```

이때 아래 코드를 실행하면:

```python
title = item.get_text()
```

`title`에는 아래 값만 들어간다.

```text
첫 번째 뉴스
```

즉, `<a>`, `class`, `href` 같은 HTML 요소는 제거되고 텍스트만 남는다.

공백을 정리하려면 아래처럼 사용한다.

```python
title = item.get_text(strip=True)
```

## 7. 태그에서 링크 가져오기

실제 크롤링에서는 글자뿐만 아니라 링크도 자주 가져온다.

예시 HTML:

```html
<a class="news_tit" href="https://news.example.com/1">
  첫 번째 뉴스
</a>
```

텍스트 가져오기:

```python
title = item.get_text(strip=True)
```

결과:

```text
첫 번째 뉴스
```

링크 가져오기:

```python
link = item["href"]
```

결과:

```text
https://news.example.com/1
```

더 안전하게 가져오려면 아래처럼 쓴다.

```python
link = item.get("href")
```

`href` 속성이 없는 태그를 만나도 에러가 덜 난다.

## 8. 최종 예시 코드

아래 코드는 정적 페이지에서 `a` 태그의 텍스트와 링크를 수집해서 CSV로 저장하는 예시다.

```python
import requests
from bs4 import BeautifulSoup
import pandas as pd

url = "https://www.naver.com"

response = requests.get(url)
response.raise_for_status()

html_text = response.text
soup = BeautifulSoup(html_text, "html.parser")

items = soup.select("a")

rows = []

for item in items:
    text = item.get_text(strip=True)
    link = item.get("href")

    rows.append({
        "text": text,
        "link": link,
    })

df = pd.DataFrame(rows)

df.columns = df.columns.astype(str).str.strip()
df = df.dropna(how="all")
df = df.drop_duplicates()

print(df.head())
print(df.info())
print(df.isnull().sum())

df.to_csv("data/processed/static_page.csv", index=False, encoding="utf-8-sig")
```

## 10. 기본 코드 템플릿

실제 프로젝트에서는 아래 코드를 복사한 뒤 `url`, `CSS_SELECTOR`, 컬럼명만 바꿔서 사용하면 된다.

```python
import requests
from bs4 import BeautifulSoup
import pandas as pd

url = "수집할 URL"
response = requests.get(url)
response.raise_for_status()

soup = BeautifulSoup(response.text, "html.parser")

rows = []
for item in soup.select("CSS_SELECTOR"):
    rows.append({
        "title": item.get_text(strip=True),
        "link": item.get("href"),
    })

df = pd.DataFrame(rows)
df.columns = df.columns.astype(str).str.strip()
df = df.dropna(how="all")
df = df.drop_duplicates()

df.to_csv("data/processed/static_page.csv", index=False, encoding="utf-8-sig")
```

## 초보자가 꼭 기억할 것

- `requests.get(url)`은 웹페이지에 접속해서 응답을 가져온다.
- `response.text`는 HTML 문서 전체를 문자열로 가져온다.
- `BeautifulSoup()`은 HTML을 분석하기 쉬운 형태로 바꾼다.
- `find()`는 조건에 맞는 태그를 하나만 찾는다.
- `find_all()`은 조건에 맞는 태그를 여러 개 찾는다.
- `select_one()`은 CSS 선택자로 태그를 하나만 찾는다.
- `select()`는 CSS 선택자로 태그를 여러 개 찾는다.
- `get_text()`는 태그 안의 글자만 꺼낸다.
- `.get("href")`는 링크 주소 같은 속성값을 꺼낸다.
- 최종적으로 `pandas DataFrame`으로 만들면 CSV 저장이 쉽다.

- 빈 행과 중복 행이 제거되어 있다.
