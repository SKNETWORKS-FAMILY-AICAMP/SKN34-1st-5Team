# JSON CSV 변환

## 목표

`JSON` API 또는 JSON 파일을 읽어서 `CSV` 파일로 변환한다.

JSON은 API에서 가장 많이 사용되는 데이터 형태다. 파이썬의 딕셔너리와 리스트 구조와 비슷해서 데이터를 다루기 쉽지만, 중첩 구조가 많기 때문에 실제 데이터가 어디에 들어있는지 먼저 확인하는 과정이 중요하다.

JSON CSV 변환은 보통 아래 흐름으로 진행한다.

```text
JSON API 주소 또는 JSON 파일 준비
→ requests로 JSON 응답 가져오기
→ response.json()으로 파이썬 자료형으로 변환하기
→ JSON 구조 확인하기
→ 실제 데이터 리스트 위치 찾기
→ pandas.json_normalize()로 DataFrame 만들기
→ CSV 파일로 저장하기
```

## 최종 산출물

- JSON API URL 또는 JSON 파일
- 실제 데이터가 들어있는 키 위치
- 전처리 완료된 `CSV` 파일
- 데이터 출처와 수집 날짜 기록

## 작업 순서

1. JSON URL 또는 JSON 파일 경로를 준비한다.
2. `requests`로 JSON 데이터를 가져온다.
3. `response.json()`으로 JSON을 파이썬 자료형으로 변환한다.
4. `type()`, `keys()`, 미리보기 출력으로 JSON 구조를 확인한다.
5. 실제 데이터가 들어있는 리스트 위치를 찾는다.
6. `pandas.json_normalize()`로 표 형태로 펼친다.
7. 컬럼명, 결측치, 중복값을 정리한다.
8. 최종 결과를 `CSV`로 저장한다.

## 1. JSON이란?

JSON은 데이터를 `{}`와 `[]`를 사용해서 표현하는 형식이다.

예시 JSON:

```json
{
  "items": [
    {
      "region": "서울",
      "count": 10000,
      "year": 2024
    },
    {
      "region": "부산",
      "count": 5000,
      "year": 2024
    }
  ]
}
```

이 JSON에는 `items`라는 키 안에 실제 데이터 리스트가 들어있다.

첫 번째 데이터:

```text
region: 서울
count: 10000
year: 2024
```

두 번째 데이터:

```text
region: 부산
count: 5000
year: 2024
```

CSV로 바꾸면 아래처럼 된다.

```csv
region,count,year
서울,10000,2024
부산,5000,2024
```

## 2. JSON의 기본 구조

JSON에서 자주 보는 구조는 크게 두 가지다.

### 딕셔너리 구조

```json
{
  "name": "서울",
  "value": 10000
}
```

파이썬에서는 딕셔너리처럼 다룬다.

```python
data["name"]
```

결과:

```text
서울
```

### 리스트 구조

```json
[
  {"name": "서울", "value": 10000},
  {"name": "부산", "value": 5000}
]
```

파이썬에서는 리스트처럼 다룬다.

```python
data[0]
```

결과:

```python
{"name": "서울", "value": 10000}
```

CSV로 바꿀 때는 보통 딕셔너리 여러 개가 들어있는 리스트가 필요하다.

## 3. requests로 JSON 가져오기

```python
import requests

url = "JSON API URL"
response = requests.get(url)
response.raise_for_status()
data = response.json()
```

### 한 줄씩 설명

```python
import requests
```

웹 API에 접속하기 위해 `requests`를 가져온다.

```python
url = "JSON API URL"
```

JSON 데이터를 제공하는 API 주소를 저장한다.

예시:

```text
https://example.com/api/items.json
```

```python
response = requests.get(url)
```

해당 URL로 요청을 보내고 응답을 받는다.

```python
response.raise_for_status()
```

요청이 실패했을 때 에러를 발생시킨다.

```python
data = response.json()
```

응답으로 받은 JSON 문자열을 파이썬 자료형으로 변환한다.

JSON이 `{}`로 시작하면 보통 파이썬 딕셔너리가 되고, `[]`로 시작하면 보통 파이썬 리스트가 된다.

## 4. JSON 구조 확인하기

JSON은 구조가 다양하기 때문에 바로 `DataFrame`으로 만들지 말고 먼저 구조를 확인해야 한다.

```python
print(type(data))
```

결과 예시 1:

```text
<class 'dict'>
```

딕셔너리 구조라는 뜻이다.

결과 예시 2:

```text
<class 'list'>
```

리스트 구조라는 뜻이다.

딕셔너리라면 어떤 키가 있는지 확인한다.

```python
print(data.keys())
```

결과 예시:

```text
dict_keys(['response', 'items', 'page'])
```

이 결과는 `data` 안에 `response`, `items`, `page`라는 키가 있다는 뜻이다.

## 5. 실제 데이터 위치 찾기

JSON API는 실제 데이터가 한 번에 보이지 않고 여러 단계 안에 들어있는 경우가 많다.

예시 JSON:

```json
{
  "response": {
    "body": {
      "items": [
        {"region": "서울", "count": 10000},
        {"region": "부산", "count": 5000}
      ]
    }
  }
}
```

이 경우 실제 데이터 리스트는 아래 위치에 있다.

```python
items = data["response"]["body"]["items"]
```

결과 예시:

```python
[
    {"region": "서울", "count": 10000},
    {"region": "부산", "count": 5000}
]
```

이 `items`가 CSV의 행 데이터가 된다.

## 6. pandas.json_normalize 사용하기

`pandas.json_normalize()`는 JSON 데이터를 표 형태로 펼쳐준다.

```python
import pandas as pd

df = pd.json_normalize(items)
```

결과 예시:

```text
  region  count
0     서울  10000
1     부산   5000
```

중첩된 JSON도 어느 정도 펼쳐준다.

예시 JSON:

```json
[
  {
    "region": "서울",
    "charger": {
      "fast": 100,
      "slow": 300
    }
  }
]
```

코드:

```python
df = pd.json_normalize(items)
```

결과 예시:

```text
  region  charger.fast  charger.slow
0     서울           100           300
```

중첩된 키는 `charger.fast`처럼 점으로 연결된 컬럼명이 된다.

## 7. JSON 파일을 직접 읽는 경우

API가 아니라 로컬 JSON 파일을 읽는 경우도 있다.

```python
import json

file_path = "data/raw/source.json"

with open(file_path, "r", encoding="utf-8") as file:
    data = json.load(file)
```

`json.load(file)`은 JSON 파일 내용을 파이썬 딕셔너리 또는 리스트로 변환한다.

## 8. DataFrame 전처리하기

```python
df.columns = df.columns.astype(str).str.strip()
```

컬럼명을 문자열로 바꾸고 앞뒤 공백을 제거한다.

```python
df = df.dropna(how="all")
```

모든 값이 비어 있는 행을 제거한다.

```python
df = df.drop_duplicates()
```

완전히 같은 중복 행을 제거한다.

```python
print(df.info())
```

컬럼명, 데이터 개수, 데이터 타입을 확인한다.

```python
print(df.isnull().sum())
```

컬럼별 결측치 개수를 확인한다.

## 9. 최종 예시 코드

```python
import requests
import pandas as pd

url = "JSON API URL"

response = requests.get(url)
response.raise_for_status()

data = response.json()

print("data 타입:", type(data))

if isinstance(data, dict):
    print("최상위 키:", data.keys())
else:
    print("리스트 데이터 개수:", len(data))

# 실제 데이터 위치는 JSON 구조에 맞게 수정해야 한다.
# 예시 1: data 자체가 리스트인 경우
# items = data

# 예시 2: data["items"]에 실제 데이터가 있는 경우
# items = data["items"]

# 예시 3: data["response"]["body"]["items"]에 실제 데이터가 있는 경우
# items = data["response"]["body"]["items"]

items = data

df = pd.json_normalize(items)

df.columns = df.columns.astype(str).str.strip()
df = df.dropna(how="all")
df = df.drop_duplicates()

print(df.head())
print(df.info())
print(df.isnull().sum())
print(df.duplicated().sum())

df.to_csv("data/processed/json_data.csv", index=False, encoding="utf-8-sig")
```

## 11. 자주 나는 오류

### JSONDecodeError

```text
JSONDecodeError
```

원인:

- 응답이 JSON이 아닐 수 있다.
- API 오류 메시지가 HTML 또는 XML로 내려왔을 수 있다.
- 인증키가 잘못되었을 수 있다.

확인 코드:

```python
print(response.status_code)
print(response.text[:500])
```

### KeyError

```text
KeyError: 'items'
```

원인:

- `items`라는 키가 실제로 없을 수 있다.
- JSON 구조가 예상과 다를 수 있다.

해결:

```python
print(data.keys())
```

최상위 키부터 하나씩 확인한다.

### DataFrame이 이상하게 한 줄만 나오는 경우

원인:

- 실제 데이터 리스트가 아니라 전체 딕셔너리를 바로 `json_normalize()` 했을 수 있다.

해결:

- 실제 데이터가 들어있는 리스트 위치를 찾아서 그 리스트를 `json_normalize()`에 넣는다.

```python
items = data["response"]["body"]["items"]
df = pd.json_normalize(items)
```

## 12. 기본 코드 템플릿

실제 프로젝트에서는 아래 코드에서 `url`과 `items = ...` 부분을 JSON 구조에 맞게 수정하면 된다.

```python
import requests

response.raise_for_status()

print(data.keys() if isinstance(data, dict) else "list data")

```

## 초보자가 꼭 기억할 것

- JSON은 `{}` 딕셔너리와 `[]` 리스트 구조로 이루어진다.
- `response.json()`은 JSON 응답을 파이썬 자료형으로 바꾼다.
- `type(data)`로 딕셔너리인지 리스트인지 먼저 확인한다.
- `data.keys()`로 어떤 키가 있는지 확인한다.
- 실제 데이터 리스트 위치를 찾아야 CSV로 제대로 변환할 수 있다.
- `pd.json_normalize()`는 JSON을 표 형태로 펼쳐준다.
- 중첩된 키는 `상위키.하위키` 형태의 컬럼명으로 변환될 수 있다.
