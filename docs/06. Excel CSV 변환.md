# Excel CSV 변환

## 목표

`xlsx`, `xls` 형태의 Excel 파일을 읽어서 `CSV` 파일로 변환한다.

공공기관 데이터는 Excel 파일로 제공되는 경우가 많다. Excel 파일은 시트가 여러 개일 수 있고, 상단에 제목이나 설명 행이 들어있는 경우도 많다. 그래서 바로 CSV로 저장하기보다 시트 목록, 실제 데이터 시작 행, 컬럼명을 먼저 확인해야 한다.

Excel CSV 변환은 보통 아래 흐름으로 진행한다.

```text
Excel 파일을 data/raw 폴더에 저장
→ pandas.ExcelFile()로 시트 목록 확인
→ pandas.read_excel()로 사용할 시트 읽기
→ 상단 설명 행과 빈 행 정리
→ 컬럼명과 중복값 정리
→ CSV 파일로 저장하기
```

## 최종 산출물

- 원본 Excel 파일
- 사용할 시트명 또는 시트 번호
- 전처리 완료된 `CSV` 파일
- 데이터 출처와 수집 날짜 기록

## 작업 순서

1. Excel 파일을 `data/raw/` 폴더에 저장한다.
2. 파일 확장자가 `xlsx`인지 `xls`인지 확인한다.
3. `pandas.ExcelFile()`로 시트 목록을 확인한다.
4. 사용할 시트를 선택한다.
5. `pandas.read_excel()`로 시트를 읽는다.
6. 불필요한 상단 설명 행이 있으면 `skiprows`로 제거한다.
7. 컬럼명, 빈 행, 중복값, 결측치를 정리한다.
8. 최종 결과를 `CSV`로 저장한다.

## 1. Excel 파일 준비하기

원본 Excel 파일은 `data/raw/` 폴더에 저장한다.

예시 폴더 구조:

```text
data/
├── raw/
│   └── ev_charger_status.xlsx
└── processed/
```

원본 파일과 전처리 결과 파일을 분리해두면 나중에 문제가 생겼을 때 원본을 다시 확인하기 쉽다.

## 2. pandas로 Excel 파일 읽기

Excel 파일을 읽으려면 보통 `pandas`를 사용한다.

설치가 안 되어 있다면 아래 패키지가 필요할 수 있다.

```bash
pip install pandas openpyxl
```

`openpyxl`은 `.xlsx` 파일을 읽을 때 자주 사용된다.

## 4. 특정 시트 읽기

시트 번호로 읽을 수도 있고 시트 이름으로 읽을 수도 있다.

### 시트 번호로 읽기

```python
df = pd.read_excel(file_path, sheet_name=0)
```

`sheet_name=0`은 첫 번째 시트를 의미한다.

주의할 점은 파이썬 번호는 0부터 시작한다.

```text
sheet_name=0 → 첫 번째 시트
sheet_name=1 → 두 번째 시트
sheet_name=2 → 세 번째 시트
```

### 시트 이름으로 읽기

```python
df = pd.read_excel(file_path, sheet_name="충전소 현황")
```

시트 이름을 정확히 알고 있다면 이름으로 읽는 것이 더 명확하다.

## 5. 데이터 미리보기

Excel 파일을 읽은 뒤에는 바로 저장하지 말고 앞부분을 확인한다.

```python
print(df.head())
```

결과 예시:

```text
      전기차 충전소 현황 Unnamed: 1 Unnamed: 2
0          기준일: 2024년      NaN      NaN
1                  지역    충전소수     기준연도
2                  서울      1000     2024
3                  부산       500     2024
```

이런 경우 실제 컬럼명은 1번 행에 있고, 위쪽에는 제목 행이 섞여 있다.

## 6. 상단 설명 행 제거하기

상단에 제목이나 설명 행이 있으면 `skiprows`를 사용한다.

```python
df = pd.read_excel(file_path, sheet_name="충전소 현황", skiprows=2)
```

`skiprows=2`는 위에서 2줄을 건너뛰고 읽는다는 뜻이다.

예를 들어 원본이 아래와 같다면:

```text
0행: 전기차 충전소 현황
1행: 기준일: 2024년
2행: 지역 | 충전소수 | 기준연도
3행: 서울 | 1000 | 2024
```

`skiprows=2`를 쓰면 2행부터 컬럼으로 읽기 시작한다.

결과 예시:

```text
   지역  충전소수  기준연도
0  서울  1000  2024
1  부산   500  2024
```

## 7. 컬럼명 정리하기

Excel 파일을 읽으면 컬럼명에 공백이 있거나 `Unnamed` 컬럼이 생기는 경우가 있다.

```python
df.columns = df.columns.astype(str).str.strip()
```

컬럼명을 문자열로 바꾸고 앞뒤 공백을 제거한다.

예시:

```text
" 지역 " → "지역"
```

불필요한 `Unnamed` 컬럼을 제거할 수도 있다.

```python
df = df.loc[:, ~df.columns.str.contains("^Unnamed")]
```

의미:

```text
컬럼명 중 Unnamed로 시작하는 컬럼은 제거한다.
```

## 8. 빈 행과 중복 행 제거하기

```python
df = df.dropna(how="all")
```

모든 값이 비어 있는 행을 제거한다.

```python
df = df.drop_duplicates()
```

완전히 같은 중복 행을 제거한다.

```python

```

컬럼별 결측치 개수를 확인한다.

```python

```

중복 행 개수를 확인한다.

## 10. 최종 예시 코드

```python
import pandas as pd

file_path = "data/raw/source.xlsx"

excel_file = pd.ExcelFile(file_path)
print("시트 목록:", excel_file.sheet_names)

sheet_name = 0

df = pd.read_excel(file_path, sheet_name=sheet_name)

print("원본 미리보기")
print(df.head())

# 상단 설명 행이 있으면 아래 코드처럼 skiprows를 조정해서 다시 읽는다.
# df = pd.read_excel(file_path, sheet_name=sheet_name, skiprows=2)

df.columns = df.columns.astype(str).str.strip()
df = df.loc[:, ~df.columns.str.contains("^Unnamed")]
df = df.dropna(how="all")
df = df.drop_duplicates()

print("전처리 후 미리보기")
print(df.head())
print(df.info())
print(df.isnull().sum())
print(df.duplicated().sum())

df.to_csv("data/processed/excel_data.csv", index=False, encoding="utf-8-sig")
```

## 12. 자주 나는 오류

### Missing optional dependency openpyxl

```text
Missing optional dependency 'openpyxl'
```

원인:

- `.xlsx` 파일을 읽기 위한 `openpyxl` 패키지가 설치되어 있지 않다.

해결:

```bash
pip install openpyxl
```

### FileNotFoundError

```text
FileNotFoundError
```

원인:

- `file_path` 경로가 잘못되었다.
- 파일이 `data/raw/` 폴더에 없다.
- 파일명 또는 확장자가 다르다.

해결:

- 실제 파일 위치를 확인한다.
- 파일명에 공백이나 한글이 있으면 정확히 입력한다.

### 컬럼명이 이상하게 나오는 경우

예시:

```text
Unnamed: 0, Unnamed: 1
```

원인:

- 실제 컬럼명 위에 제목이나 설명 행이 있다.
- `skiprows` 설정이 필요하다.

해결:

```python
df = pd.read_excel(file_path, sheet_name=0, skiprows=2)
```

## 13. 기본 코드 템플릿

```python
import pandas as pd

file_path = "data/raw/source.xlsx"
```

## 초보자가 꼭 기억할 것

- Excel 파일은 시트가 여러 개일 수 있으므로 `sheet_names`를 먼저 확인한다.
- `sheet_name=0`은 첫 번째 시트를 의미한다.
- 상단 설명 행이 있으면 `skiprows`로 건너뛴다.
- `Unnamed` 컬럼은 불필요한 빈 컬럼일 가능성이 높다.
- 컬럼명, 빈 행, 중복 행을 정리한 뒤 CSV로 저장한다.
- 한글 CSV는 `encoding="utf-8-sig"`로 저장하면 엑셀에서 깨질 가능성이 줄어든다.
